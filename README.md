# Ethiopian Vital Events Management System

A comprehensive web application for managing birth, death, marriage, and divorce records in Ethiopia.

## Features

- ✅ Death Records Management
- ✅ Marriage Records Management  
- ✅ Divorce Records Management
- ✅ User Authentication & Authorization
- ✅ Ethiopian Date Support
- ✅ Certificate Number Generation
- ✅ Role-based Access Control

## Tech Stack

- **Backend**: Flask, Python
- **Database**: MongoDB
- **Authentication**: JWT
- **API Testing**: Postman
